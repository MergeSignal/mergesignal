# Support and feedback

Thank you for your interest in MergeSignal. This repository is public under Apache 2.0.

MergeSignal is maintained with a deliberate product direction. The repository is public, but the product is not community-governed.

Feature ideas are welcome through GitHub issues, discussions, feature requests, or [Contact](https://mergesignal-web.fly.dev/contact). Product direction, UX, scoring methodology, roadmap decisions, and architecture remain maintainer-driven. Direct implementation of product features is not expected.

## What we welcome

- Bug reports
- Feature requests
- Documentation improvement suggestions
- Build and tooling improvement suggestions
- Questions and feedback

## How to reach us

- **GitHub issues** — bugs, feature requests, and suggestions
- **[Contact](https://mergesignal-web.fly.dev/contact)** on the hosted web app
